# Kcalc
Питон версия простейшего калькулятора __Kcalc__ c расширенными функциями.\
На данный момент существуют функции:
* сложение
* вычитание
* умножение
* деление
  * целочисленное
  * остаток
  * обычное
* степень числа
* корень
* факториал

Скачать последнюю [версию](https://github.com/MrPart/Kcalc/releases).
